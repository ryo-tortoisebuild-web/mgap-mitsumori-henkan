# 見積書 → 工事内容概要書 変換ツール（MGAP）

見積書PDFを読み込み、金額情報を除いた「工事内容概要書」を **Word（.docx）** で書き出すツール。
職人・大工向けの作業指示書として使うため、単価・金額・諸経費・値引き・定価は一切出力しない。

## 使い方
1. ブラウザでツールを開く
2. 見積書PDFをドラッグ＆ドロップ
3. 表示された表を必要に応じてその場で修正
4. 「Wordでダウンロード」→ Wordで追記 → Wordの「PDFとして保存」

## 出力仕様
| 列 | 内容 |
|---|---|
| 大項目 | 工種（＋小分類）例：`設備工事（ユニットバス）` |
| 小項目 | 見積書の「項目」をそのまま転記 |
| 概要・摘要 | 見積書の「規格・摘要」＋数量（**一式以外のみ**記載） |
| 備考 | 見積書の「備考」をそのまま転記（金額表現のみ除去） |

### 自動で除外されるもの
- 単価・金額・小計・合計
- 「その他」工種（駐車場代・現場管理費・諸経費）
- 値引き
- 定価・参考価格などの金額表現

## 構成
`index.html` 1ファイルのみ。サーバー側の処理はなし。
PDFの解析はすべてブラウザ内で完結し、**見積書は外部に送信されない**。

依存（CDN読み込み）:
- pdf.js 3.11.174 … PDFのテキスト＋座標抽出
- docx 8.5.0 … .docx生成

## 設置方法
`index.html` をWebサーバーの公開ディレクトリに置くだけ。担当者にはURLを共有する。

## 注意
明細の抽出は見積書テンプレートのレイアウト（列のx座標）に依存している。
見積書のフォーマットが変わった場合は `index.html` 内の `COLS` の座標調整が必要。
