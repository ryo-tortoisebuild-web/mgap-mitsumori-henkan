# 見積書 → 工事内容概要書 変換ツール

リフォーム工事の見積書PDFを読み込み、工事内容の一覧を **Word（.docx）** として書き出すブラウザツール。
現場向けの資料として使うため、金額に関する情報は出力しない。

## 使い方

1. ブラウザでツールを開く
2. 見積書PDFをドラッグ＆ドロップ
3. 表示された表を必要に応じてその場で修正
4. 「Wordでダウンロード」→ Wordで追記 → Wordの「PDFとして保存」

## 出力仕様

| 列 | 内容 |
| --- | --- |
| 大項目 | 工種（＋小分類）例：`設備工事（ユニットバス）` |
| 小項目 | 見積書の「項目」をそのまま転記 |
| 概要・摘要 | 見積書の「規格・摘要」＋数量（**一式以外のみ**記載） |
| 備考 | 見積書の「備考」をそのまま転記 |

金額に関する情報（単価・金額・小計・合計・定価など）は出力されない。

## 構成

`index.html` 1ファイルのみ。サーバー側の処理はなし。
PDFの解析はすべてブラウザ内で完結し、**読み込んだPDFが外部に送信されることはない**。

依存（CDN読み込み）:

- [pdf.js](https://mozilla.github.io/pdf.js/) 3.11.174 … PDFのテキスト＋座標抽出
- [docx](https://docx.js.org/) 8.5.0 … .docx生成

## 設置方法

`index.html` を静的ホスティングに置くだけ（GitHub Pages / 任意のWebサーバー）。

## 実装メモ

明細の抽出は、PDF内の各テキストのx座標から列を判定するルールベース方式。
対応する見積書のレイアウトは `index.html` 内の `COLS` で定義しているため、
レイアウトの異なる見積書に対応させる場合はこの座標の調整が必要。
